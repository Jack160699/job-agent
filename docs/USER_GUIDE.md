# User Guide

## Getting Started

### 1. Create Account
Sign up at `/signup` with your email and password.

### 2. Upload Master Resume
1. Go to **Dashboard → Resumes**
2. Paste your full resume text
3. Click **Upload Resume**
4. Skills are automatically detected

### 3. Configure Search Filters
1. Go to **Dashboard → Settings**
2. Set job titles, locations, experience, salary range
3. Set match threshold (default: 70%)
4. Choose enabled job sources
5. Save settings

### 4. Search for Jobs
1. Go to **Dashboard → Job Search**
2. Click **Search Jobs**
3. Jobs appear from configured sources
4. Each job gets an application record automatically

### 5. Review AI Matches
1. Go to **Dashboard → AI Matches**
2. Jobs are ranked by match score
3. Review strengths and gaps for each match
4. Scores below threshold are auto-skipped

### 6. Review Applications
1. Go to **Dashboard → Applications**
2. Applications with status **PENDING_REVIEW** need your approval
3. Review tailored resume and cover letter
4. Approve or reject before submission

## Automation Settings

### Review Before Submit (Recommended)
When enabled, the agent pauses before final submission so you can review tailored documents.

### Auto-Submit
Only enable for trusted ATS platforms (Greenhouse, Lever, Ashby). Never auto-submit on LinkedIn or Indeed.

### Match Threshold
Jobs below this score are automatically skipped. Recommended: 70% for quality focus.

## Tips for Best Results

1. **Keep your master resume comprehensive** — the AI can only use what's there
2. **Set specific job titles** — "Senior Frontend Engineer" beats "Engineer"
3. **Review before submitting** — always check tailored documents
4. **Update skills regularly** — re-upload resume when you learn new skills
5. **Use analytics** — track which sources yield the best matches
